## 3. 学到的新知识
- 小米 MiMo TTS API 使用 `api.xiaomimimo.com/v1/chat/completions`，model 为 `mimo-v2-tts`
- 飞书发语音需要 ogg 格式文件，且文件需在 workspace 目录下（allowed mediaLocalRoots）
- PWA 可以替代 APK 作为安卓应用安装方案，无需编译环境
- 苹果设计语言核心元素：毛玻璃模糊（backdrop-filter: blur）、0.5px 边框、极轻阴影、渐变背景
- CSS columns 实现瀑布流布局比手动计算更优雅
- 子代理（sub-agent）处理大任务时可能只做一半，需要主 agent 兜底完成
